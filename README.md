# AI DỰ BÁO THỊ TRƯỜNG CHỨNG KHOÁN

## Giới thiệu
Đây là dự án đồ án môn học xây dựng hệ thống dự báo thị trường chứng khoán bằng trí tuệ nhân tạo. Hệ thống giúp phân tích dữ liệu lịch sử, dự đoán xu hướng giá cổ phiếu và hỗ trợ người dùng trong việc ra quyết định đầu tư.

## Chức năng chính
* Thu thập và xử lý dữ liệu chứng khoán
* Phân tích dữ liệu lịch sử
* Xây dựng và huấn luyện mô hình dự báo
* Dự đoán xu hướng giá cổ phiếu
* Hiển thị kết quả dự đoán trực quan
* Hỗ trợ người dùng theo dõi và đánh giá thị trường

## Công nghệ sử dụng
* Python (ngôn ngữ chính để xử lý dữ liệu và xây dựng mô hình AI)
* Anaconda (môi trường quản lý thư viện và chạy dự án)
* Antigravity (hỗ trợ xử lý logic và cấu trúc chương trình)
* Streamlit.io (xây dựng giao diện web hiển thị kết quả)
